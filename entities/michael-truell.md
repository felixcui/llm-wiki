# Michael Truell

**身份**: Cursor 创始人兼 CEO
**核心观点**: 软件开发三纪元模型（Tab → Agent → Team）

## 概述

Michael Truell 是 [[cursor]] 的创始人兼 CEO。2026年5月，他在 Cursor 新办公室发表了《AI Coding 的下一个纪元》演讲，用数据揭示了软件开发行业的时代进程，提出 Tab → Agent → Team 三纪元模型。^[raw/articles/2026-05-14_Agent请求量暴涨15倍！Cursor创始人：75%企业代码已由AI生成，内部30%PR已由Agent端到端完成，探索无人类介入的自主Team模式.md]

## 核心观点

- **工程师角色转变**：从手写代码到成为 Agent 管理者
- **"幽灵同事"概念**：成千上万名 AI Agent 和人类工程师协作
- **审查成为核心工作**：Agent 时代工程师花更多时间审查 Agent 产出
- **自主团队是未来**：Agent 团队在无人类介入下端到端完成开发
