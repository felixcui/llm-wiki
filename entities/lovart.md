# Lovart

**类型**: AI 图像设计与生成平台
**集成模型**: [[gpt-image-2]]、[[seedance-2]]

## 概述

Lovart 是一个 AI 图像设计与生成平台，深度集成了 [[gpt-image-2]] 模型，提供从设计到展示的完整创作流程。支持三种图像质量档位，并提供 Agent 批量生图、TouchEdit 局部修改、Mockup 样机、视频生成等高级功能。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

## 核心功能

### TouchEdit 精准局部修改

支持在一张包含多个元素的图中精准选中特定元素进行替换或修改，"指哪打哪"，无需重新生成整张图。适合多元素信息图的精细化修改。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

### Agent 批量生图

确定一个满意的排版样式后，可以通过 Agent 自动批量生成同系列图片，保持排版布局和信息一致性，省时省力。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

### 编辑文字功能

可以直接修改 [[gpt-image-2]] 生成图片中的文字，不改变原图其他元素，且能保证字体一致性。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

### Mockup 样机

支持创建空白产品图，将设计图直接印在空白产品上查看效果图，如抱枕、手机壳等。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

### 多角度功能

可以调整 IP 形象的查看视角，从不同角度展示设计。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

### 视频生成器

可调用 [[seedance-2]] 为设计产品创建展示动画。^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]

## 应用场景

Lovart 搭配 [[gpt-image-2]] 覆盖的 23+ 真实场景包括：

- 信息图鉴（MBTI、食物热量分析、历史事件分析）
- IP 形象设计与品牌周边
- 平面设计（开业海报、展览海报）
- UI 设计（App 界面）
- 游戏设计（角色、战斗画面、地图）
- 产品分析（构造分析、原理拆解）^[raw/articles/2026-04-24_Image2能批量生图了，23个真实场景和提示语一口气学会！.md]
