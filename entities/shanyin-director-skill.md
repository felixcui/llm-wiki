# 山音超级导演大师 Skill

**作者**: 山音（AIGC 艺术家、独立导演、编剧）
**GitHub**: [github.com/Shanyin-ai/shanyin-director-master](https://github.com/Shanyin-ai/shanyin-director-master)
**配套 Skill**: [山音超级编剧大师](https://github.com/Shanyin-ai/shanyin-screenwriting-master)
**类型**: AI 影视分镜生成 Skill

## 概述

「山音超级导演大师」是一个将专业导演流程封装进 AI 的 Claude Skill，可从一句话创意出发，经过五步工作流生成可拍摄的标准分镜表。使用 Opus 4.7 搭配，实测可在两小时内从一句话做出 62 镜、4 分 15 秒的标准分镜方案。^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]

## 五步工作流

1. **导演定调**：从内置导演方法论库中提取差异化导演方向，具体到摄影机运动、光线色调、剪辑节奏、声音处理。确定七项视听决策和一套意象体系。
2. **节奏规划**：为每场戏标注"情节节奏"和"情感节奏"两条线，绘制全片节奏波浪图。
3. **剧本微调**：将定调阶段规划的意象落到具体剧本位置，通过道具串联整条情感线。
4. **分镜拆解**：按场景分批输出分镜，每个镜头有明确叙事目的，相邻镜头之间有视觉逻辑。约 37% 为"出彩镜头"落在叙事重心上。
5. **输出 xlsx 文件**：标准 Excel 格式，冻结首行、自动换行、场景间分隔、出彩镜头星号标注，可直接打印带到片场。^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]

## 关键特性

### 逐步确认机制

每走完一步都会停下来等用户确认，保证出品和用户想法高度符合。用户可以随时要求"自检"，Skill 会重新检查当前步骤的输出。^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]

### 方法论驱动

不是简单的"给一段故事吐出分镜表"，而是背后有一套真正经过拍摄验证的导演思维体系，参考李沧东、是枝裕和、马力克、阿彼察邦、奉俊昊等导演的方法论。^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]

### 低门槛使用

输入不用很正式——一段故事梗概、大纲甚至一句话灵感都可以。不需要是专业导演或编剧。^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]

## 使用方式

1. 复制 GitHub 仓库链接发给 Claude
2. 输入"安装这个 Skill"
3. 开新对话，输入"帮我拆分镜"并粘贴故事文字
4. 每步完成后回复"通过"继续下一步^[raw/articles/2026-04-24_一句话变成62个专业分镜：山音超级导演大师Skill来了.md]
