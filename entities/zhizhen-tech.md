# 智诊科技

**类型**: 医疗 AI 公司
**核心产品**: [[wiseclaw]]（医疗 Agent OS 平台）、WiseDiag（医疗多模态大模型）

## 概述

智诊科技是一家中国医疗 AI 公司，专注于将 AI Agent 技术应用于医疗健康行业。其核心产品包括 WiseClaw 医疗 Agent OS 平台和 WiseDiag 医疗多模态大模型。^[raw/articles/2026-05-08-medical-ai-harness-era.md]

## 核心产品

### WiseDiag 医疗多模态大模型

- 千亿级参数规模，可综合理解体检报告、检验指标、医学影像、体征照片等多源健康信息
- 在 MedBench、HealthBench 位居第一
- WiseDiag-v2 在全球医学 AI 排行榜 DoctorBench 荣登榜首，超越 Google Gemini 和 OpenAI GPT-5.4

### WiseClaw 医疗 Agent OS

参见 [[wiseclaw]]。底层结合 [[openclaw]] 与 [[prompt-context-harness-engineering]] 理念，面向医疗健康行业提供可控、可追溯、可长期交付的 Agent 运行平台。

## 产业生态

- 合作 300+ 顶级三甲医院、500+ 头部医疗健康企业
- 业务覆盖医疗机构、保康养结合、智能硬件、传统医药、金融服务及健康零售
- 用户产品：好伴 AI

## 融资

2026年完成 6500 万元天使轮融资，由杭州千遇智汇、无锡元启联合领投，华睿投资、上海珺灏筠、嘉兴青于蓝新聚能及多位投资方跟投。资金主要用于 WiseDiag 能力提升、WiseClaw 生态建设、企业级场景解决方案深化以及好伴 AI 用户增长与商业化落地。^[raw/articles/2026-05-08-medical-ai-harness-era.md]
