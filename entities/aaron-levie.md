# Aaron Levie

**身份**: Box 创始人兼 CEO
**领域**: 企业 AI 转型、云计算、SaaS

## 概述

Aaron Levie 是企业内容管理平台 Box 的创始人兼 CEO，被认为是"企业 AI 转型领域最具有洞察力的人之一"。他在 2026 年 4 月的 20VC 播客中系统阐述了对 AI 竞争、就业、软件行业变革等多维度的判断。^[raw/articles/2026-04-23_软件正走向无界面化！旧金山创始人：智能体操作员将创造几十万岗位！AI不会取代人类！认同黄仁勋立场，硅谷在被中国开源模型反向赋能.md]

## 核心观点

### AI 与就业

Aaron 与行业主流观点（如 Anthropic CEO Dario Amodei 预测的失业潮）有分歧：
- "我们太短视、太以自我为中心，总觉得整个行业就是科技行业"——科技行业只占经济的 8%-15%
- 自动化会暴露下一层瓶颈，而非消灭岗位（律师案例：AI 生成法律内容越容易，需要审核的真正律师越多）
- 80% 认同黄仁勋的务实判断，反对"生死竞赛"叙事

### 智能体操作员（Agent Operator）

预测将出现 50-100 万个"智能体操作员"新岗位：
- 需要懂 MCP、CLI，会编写 [[agent-skill-specification|Skills]]，理解 Agent 架构
- 核心价值：进入企业各部门（市场、法务、运营、生命科学），帮助真正用上智能体
- "像手机 App 一样的软件是有限的，但需要构建的后台系统级软件是无限的"

### 软件无界面化（Headless-First）

- "软件正在走向无界面化"——Agent 在工具调用、跨系统搜索和准确性上的能力提升太快
- 软件平台必须"headless 优先"，用户界面可能只是一个和智能体对话的窗口
- 价值从 UI 层转移到 API 层——"你的 API 有多强"比"你的按钮有多少"更重要
- SaaS 不会因 AI 消失，但"价值的位置"会发生转移

这一判断被 Salesforce Headless 360 的发布和 Ramp MCP 用户 10 倍增长所验证，详见 [[agent-first-design]]。^[raw/articles/2026-04-26-designing-products-for-agents.md]

### Token 经济

- Token 预算将从 IT 支出转移到常规运营支出
- 企业需要按价值分配 token（前 5% 高价值用户配最好模型，其余用性价比模型）
- 现实世界有预算约束，不可能"全公司无限 token"

### AI 竞争格局

- OpenAI vs Anthropic：市场巨大无比，公司不会单供应商绑定，"需要一个多供应商、多 AI 的世界"
- 中国开源模型"反向赋能"硅谷的说法某种程度上成立，但不必恐惧
- AI 在编程上的效果很难被迅速复刻到其他知识工作领域
- 继续重仓前沿模型公司

### 技术扩散速度

- 18 个月不算有意义的时间窗口，"技术扩散的速度会比硅谷想象的更慢"
- 类比云计算扩散周期：早期增长陡峭但 20 年后规模远超预期
- 监管、合规、安全团队都需要时间适应
