# 苍何（Canghe）

AI 技术博主与开源开发者，公众号原创文章 530+ 篇，活跃于 AI Coding 和 Agent 实践社区。

## 开源项目

- **awesome-gpt-image-2**：系统性收集整理 [[gpt-image-2]] 提示词的开源项目，按分类、风格、场景组织，12 天内获得 4.2K GitHub Star。配套可视化网站 gpt-image2.canghe.ai 支持按分类检索提示词和案例。^[raw/articles/2026-05-08-gpt-image2-open-source.md]
- **CodexGuide**：面向中文开发者的 Codex 从入门到精通系统化开源教程知识库，域名 codexguide.ai。^[raw/articles/2026-05-08-gpt-image2-open-source.md]

## AI Coding 实践

苍何展示了通过 [[codex]] App 进行远程 [[vibecoding]] 的工作流：在医院陪护期间仅用手机连接 Mac Mini 上的 Codex，通过 Browser Use 自动搜集 X 上的提示词，结合 GitHub + Vercel 自动部署，实现"设置规则 → Codex 执行 → 人工确认"的半自动化流程。^[raw/articles/2026-05-08-gpt-image2-open-source.md]

## 多 Agent 组织实践

此前在 [[coordination-engineering]] 和 [[ai-native-organization]] 中记录了苍何使用 CodeBanana 平台搭建多 Agent 协作跨境电商公司的实践，3 人 + 5 个 Agent 覆盖传统 15 人团队的 80% 职能。
