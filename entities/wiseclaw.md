# WiseClaw

**类型**: 医疗健康行业 Agent OS 平台
**开发商**: [[zhizhen-tech]]（智诊科技）
**底层**: [[openclaw]]（连接与调度）+ Harness（治理与约束）

## 概述

WiseClaw 是智诊科技发布的面向医疗健康行业的 Agent OS 平台，2026年推出 2.0 版本。底层延续 [[openclaw]] 在连接与调度上的能力，上层将 [[prompt-context-harness-engineering]] 的核心理念做成系统默认值。^[raw/articles/2026-05-08-medical-ai-harness-era.md]

核心理念：OpenClaw 让 Agent "接得上、调得动、能执行"；Harness 让 Agent "跑得稳、管得住、追得回"。

## 核心能力

### 三层流水线

Agent 工作过程拆成三层清晰链路：
1. **Triage 分诊识别**：识别用户意图、服务场景和风险等级
2. **Clinical 临床执行**：在受控的数据、知识和工具范围内生成候选方案
3. **Evaluator 校验拦截**：通过确定性规则、医学红线和业务门禁对输出把关

关键节点可随时插入人工复核和审批。

### 健康档案驱动

将健康档案作为长期服务的核心基础，解决"每次都从头开始"的问题：
- 客观信息（检验值、体检数据、用药史、诊断结论）：确定性、受控方式读写
- 服务信息（依从性、沟通偏好、生活习惯）：结构化沉淀，持续更新

### 心跳引擎

从会话驱动升级为时间、事件和数据共同驱动：
- 用户指标异常时主动触发提醒
- 复查时间临近时服务流程自动唤醒
- 慢病指标连续波动时发起风险提示
- 长期未完成健康任务时进行干预触达

### 全链路可观测

对话、工具调用、知识引用、版本信息、流程节点、风险判断等信息结构化记录，形成完整 Trace。支持运行看板、风险审批、人机协同门禁、审计回放。

## 应用场景

WiseClaw 聚焦院外高频场景：

| 场景 | 说明 |
|------|------|
| 名医 AI 分身 | 专家诊疗逻辑 + 健康档案 + 长期记忆，H5/小程序/App 多终端 |
| 体检报告解读 | 检前问询 → 检中提醒 → 检后解读 → 历年趋势对比 |
| 健康硬件整合 | 多源数据路由器，将设备孤立数据编织成完整健康上下文 |
| 慢病营养管理 | 饮食识别 + 疾病背景 + 健康档案 + 用户偏好串联 |
| 银发家庭健康中台 | 数字家庭医生 + 适老化陪伴 + 风险预警 + 长期健康管理 |

## SKILL 模块化

将体检报告解读、慢病随访、营养干预、异常指标提醒、健康问答、风险分流、复查提醒等能力沉淀为可装配、可复用的 SKILL 模块，内置审批、脱敏、证据链、医学红线和风控策略。

## 产业生态

- 合作 300+ 顶级三甲医院、500+ 头部医疗健康企业
- 业务纵深覆盖医疗机构、保康养结合、智能硬件、传统医药、金融服务及健康零售
- 2026年完成 6500 万元天使轮融资（杭州千遇智汇、无锡元启联合领投）

## 底层模型

以智诊科技自研的千亿级 WiseDiag 医疗多模态大模型为核心基座，在 MedBench、HealthBench 位居第一，在 DoctorBench 全球医学 AI 排行榜荣登榜首（超越 Google Gemini 和 OpenAI GPT-5.4）。^[raw/articles/2026-05-08-medical-ai-harness-era.md]
