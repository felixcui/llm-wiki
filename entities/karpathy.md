# Andrej Karpathy

Andrej Karpathy 是 AI 领域最具影响力的实践者和传播者之一。[[openai]] 联合创始人、前 Tesla Autopilot VP、Eureka Labs 创始人。

## 核心贡献

### Vibe Coding
Karpathy 在 2025 年初提出了 **Vibe Coding** 概念——用自然语言描述意图，让 AI 生成代码，开发者只需"感受"结果并迭代。这一概念迅速成为 AI 编程文化的标志。

### Agentic Engineering & Software 3.0
在 2026 年 Sequoia 访谈中，Karpathy 将 Vibe Coding 升级为更系统的框架：

- **Software 3.0**：编程范式从"写代码"转向"写规格"。AI Agent 执行规格，人类负责设计和验证。
- **Agentic Engineering**：工程师的核心技能从编码变为设计 Agent 工作流、编写精确的提示词、管理系统状态。
- **AI 作为新的"解释器"**：就像 Python 取代 C 让开发者不必手动管理内存，AI Agent 取代手写代码让开发者不必管理语法。

> "编程的本质没有消失——它从写指令变成了写意图。"

## 核心观点

1. **编程不会消失，但会变**：门槛大幅降低，抽象层级提升
2. **提示词工程是过渡态**：最终目标是自然语言规格
3. **Agent 编排是核心竞争力**：未来工程师的价值在于设计多 Agent 协作系统
4. [[claude-code]] 等工具代表了从"辅助编码"到"自主编码"的范式转移
