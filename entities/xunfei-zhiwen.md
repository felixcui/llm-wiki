# 讯飞智文 (iFlytek AI PPT)

**类型**: AI PPT 生成平台
**开发商**: 科大讯飞 (iFlytek)
**用户规模**: 1000 万+

## 概述

讯飞智文是科大讯飞推出的 AI PPT 产品，采用多 Agent 架构（意图理解→大纲生成→内容填充→视觉设计），提供"写练演"全流程的演示文稿创作体验。产品标志着 AI PPT 进入"2.0 时代"，不仅生成 PPT，更覆盖从写作、练习到答辩的全链路。^[raw/articles/2026-05-07-aippt-no-rework.md]

## 核心架构

### 多 Agent 流水线

智文采用四阶段多 Agent 架构：

1. **意图理解 Agent** — 解析用户需求，确定主题与风格
2. **大纲生成 Agent** — 构建演示逻辑与内容骨架
3. **内容填充 Agent** — 生成正文、数据、案例等
4. **视觉设计 Agent** — 应用排版、配色、图文布局

### Vision Agent 模式

支持视觉理解能力，可分析图片、截图等视觉素材并融入 PPT 生成流程。^[raw/articles/2026-05-07-aippt-no-rework.md]

## 核心功能

### "写练演"全流程

- **写**: AI 自动生成 PPT 内容
- **练**: 基于生成的 PPT 进行演讲练习
- **演**: 支持数字人播报演示

### 语义驱动图像生成

通过语义理解自动为 PPT 页面生成匹配的配图，无需手动搜索素材。^[raw/articles/2026-05-07-aippt-no-rework.md]

### 答辩模拟

针对学术答辩场景，AI 可模拟答辩过程，帮助用户提前准备问题和回答。^[raw/articles/2026-05-07-aippt-no-rework.md]

### 数字人播报

支持数字人作为 PPT 演示者，实现自动化演示播报。^[raw/articles/2026-05-07-aippt-no-rework.md]
