# 张心皓 (Zhang Xinhao)

张心皓是 AI 创业公司 [[vitaflow]]（维塔流动）的创始人。曾任字节跳动核心产品高管，后担任阶跃星辰（Seedance）C 端产品合伙人，深度参与大模型 C 端产品的从 0 到 1。

## 创业方向

在多数创业者选择 B 端 AI 工作流和企业智能体赛道时，张心皓选择了消费级 AI Agent 方向，创立维塔流动并推出 [[jovida]] 产品，致力于消除个人用户从愿望到行动之间的摩擦。
