# Life Agent (消费级个人生活 AI Agent)

Life Agent 是一种面向 C 端消费者、专注于个人生活场景的 AI Agent 形态。与 B 端企业智能体和工作流 Agent 不同，Life Agent 的核心目标是消除个人用户从愿望到行动之间的摩擦（Zero Friction from Desire to Action）。

## 核心特征

- **主动式** — 不是被动等待用户提问，而是在合适的时机主动推送提醒与行动建议
- **上下文感知** — 整合日历、地点、设备信号、用户主动输入等多维信息来理解用户当前状态
- **目标拆解** — 将用户的模糊愿望/灵感（视频、照片、链接等）拆解为具体可执行步骤
- **长期记忆** — 持续学习用户习惯、偏好与节奏，形成个性化适配
- **即时收益机制** — 通过分享目标后立刻获得计划等即时反馈，激励用户主动提供信息

## 覆盖场景

健康管理、习惯养成、学习成长、社交关系、日常生活规划等个人生活全场景。

## 代表产品

- [[jovida]] — [[vitaflow]]（维塔流动）开发的 Life Agent 产品

## 与 B 端 Agent 的对比

| 维度 | Life Agent | B 端企业 Agent |
|------|-----------|---------------|
| 用户 | 个人消费者 | 企业员工/团队 |
| 目标 | 消除愿望到行动的摩擦 | 提升工作效率/自动化流程 |
| 交互 | 主动提醒 + 被动响应 | 任务驱动 + 工作流编排 |
| 上下文 | 个人生活全局（日历/位置/习惯/情绪） | 企业数据（工单/文档/系统） |
| 壁垒 | Context 积累 + 用户信任 | 系统集成 + 权限管理 |
